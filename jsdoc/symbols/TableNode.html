translateStudentWork(studentWork)
This function is called if there needs to be any special translation of the student work from the way the student work is stored to a human readable form. For example if the student work is stored as an array that contains 3 elements, for example ["apple", "banana", "orange"] and you wanted to display the student work like this Answer 1: apple Answer 2: banana Answer 3: orange you would perform that translation in this function.
